# Typography

---

### Headings

Heading h1 through h3 classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

| Tags          | Class         | Markdown      |
| ------------- | ------------- | ------------- |
| `<h1></h1>`   | `class="h1"`  | `#`           |
| `<h2></h2>`   | `class="h2"`  | `##`          |
| `<h3></h3>`   | `class="h3"`  | `###`         |

### Paragraphs

| Tags          | Class         | Markdown      |
| ------------- | ------------- | ------------- |
| `<p></p>`     | —             | —             |

### Inline text

| Tags                | Class             | Markdown                       |
| ------------------- | ----------------- | ------------------------------ |
| `<a></a>`           | `class="link"`    | \[Link name](/link/to "Link")  |
| `<code></code>`     | `class="code"`    | \`code\`                       |
| `<strong></strong>` | `class="strong"`  | \**strong\**                   |
| `<em></em>`         | `class="em"`      | \*emphasis\*                   |
| `<small></small>`   | `class="small"`   | —                              |

### Blockquotes

| Tags                            | Class                            | Markdown                       |
| ------------------------------- | -------------------------------- | ------------------------------ |
| `<blockquote></blockquote>`     | `class="blockquote"`             | \> Blockquote text             |

### List

| Tags                            | Class                            | Markdown                       |
| ------------------------------- | -------------------------------- | ------------------------------ |
| `<ul><li></li></ul>`            | `class="list"`                   | \- Link item                   |
