# Typography

---

Charm UI includes basic global styles and text utility classes. Read more about utility classes on [Utilities page](/components/utilities "Utilities")

### Headings

Heading `h1` through `h3` classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

| Tag           | Class         | Markdown      |
| ------------- | ------------- | ------------- |
| `<h1></h1>`   | `class="h1"`  | `#`           |
| `<h2></h2>`   | `class="h2"`  | `##`          |
| `<h3></h3>`   | `class="h3"`  | `###`         |

### Lead

Use `.lead` class to make Lead component.

| Tag           | Class             | Markdown      |
| ------------- | ----------------- | ------------- |
| —             | `class="lead"`    | —             |

### Inline text

Usage of common inline text elements.

| Tag                 | Class             | Markdown                       |
| ------------------- | ----------------- | ------------------------------ |
| `<a></a>`           | `class="link"`    | `[Link name](/link/to "Link")` |
| `<code></code>`     | `class="code"`    | `` `code` ``                       |
| `<strong></strong>` | `class="strong"`  | `**strong\**`                  |
| `<em></em>`         | `class="em"`      | `*emphasis*`                   |
| `<small></small>`   | `class="small"`   | —                              |

### Blockquotes

Use `.blockquote` class to make Blockquote component.

| Tag                             | Class                            | Markdown                       |
| ------------------------------- | -------------------------------- | ------------------------------ |
| `<blockquote></blockquote>`     | `class="blockquote"`             | `> Blockquote text`            |

### List

Use `.list` class to make List component.

| Tag                             | Class                            | Markdown                       |
| ------------------------------- | -------------------------------- | ------------------------------ |
| `<ul><li></li></ul>`            | `class="list"`                   | `- Link item`                  |
