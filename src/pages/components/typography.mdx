# Typography

---

### Headings

```html
<h1 class="h1">Heading</h1>
<h2 class="h2">Heading</h1>
<h3 class="h3">Heading</h1>
```

### Paragraphs

```html
<p>Vivamus sagittis lacus vel augue laoreet rutrum faucibus dolor auctor. Duis mollis, est non commodo luctus.</p>
```

### Lead

```html
<span class="lead">Vivamus sagittis lacus vel augue laoreet rutrum faucibus dolor auctor. Duis mollis, est non commodo luctus.</span>
```

### Inline text

```html
<a class="link" href="#">Links</a>
<code class="code">Code</code>
<strong class="strong">Links</strong>
<em class="em">Links</em>
<small class="small">Links</small>
```

### Blockquotes

```html
<blockquote class="blockquote">Vivamus sagittis lacus vel augue laoreet rutrum faucibus dolor auctor. Duis mollis, est non commodo luctus.</blockquote>
```

### List

```html
<ul class="list">
  <li>First item</li>
  <li>
    <ul class="list">
      <li>First inner item</li>
    </ul>
  </li>
  <li>Third item</li>
</ul>
```
